## 2. Клавіатурне керування

- **Клавіші:** `ArrowUp`, `ArrowDown`, `ArrowLeft`, `ArrowRight`
- **Швидкість:** 20 px за один натиск (`moveSpeed = 20`)
- **Межі:** рух зупиняється на краях вікна браузера
- **Flip:** рух вліво → `scaleX(1)`, вправо → `scaleX(-1)`
- **Повернення до idle:** через 500 мс після останнього натиску
- **Ігнорується** коли фокус на `INPUT`, `TEXTAREA`, `SELECT`

## 7. API (глобальні функції)

```js
// Ініціалізація — передати <img id="octopus">, опційно basePath для підкаталогів
window.initAssistant(octopusElement, { basePath: '../../assets/images/' })

// Показати факт / підказку (при кліку на восьминога)
window.onAssistantClick()

// Запит стану руху
window.isAssistantMoving() // → true | false
```

**Поведінка `onAssistantClick()`:**
- Кожен 3-й клік → пропонує невідвідану гру
- Якщо знайдено 1–2 секрети, кожен 5-й клік → повідомлення «Ти знайшов X/3 секретів!»
- Інакше → випадковий факт із пулу

## 10. CSS / Розміри

### `.assistant` (восьминіг)

| Брейкпоінт | Розмір | Позиція |
|------------|--------|---------|
| Mobile (< 480px) | 70×70 px | `bottom: 20px; right: 20px` |
| Tablet (480–768px) | 90×90 px | `bottom: 20px; right: 20px` |
| Desktop (> 768px) | 100×100 px | `bottom: 20px; right: 20px` |

- `position: fixed`, `z-index: var(--z-toast)`
- Hover: `scale(1.1) rotate(5deg)`, `filter: drop-shadow(...)`
- Transition: `transform 0.2s, filter 0.2s`

### `.fact-box` (хмарка з фактом)

| Брейкпоінт | Max-width | Позиція |
|------------|-----------|---------|
| Mobile | 250 px | `bottom: 110px; right: 10px` |
| Tablet | 280 px | `bottom: 120px; right: 15px` |
| Desktop | 300 px (50% вікна) | `bottom: 140px; right: 20px` |

- Glassmorphism: `background: rgba(...)`, `backdrop-filter: blur(...)`
- Анімація появи: `slideIn 0.4s ease`
- Приховується класом `.hidden`
