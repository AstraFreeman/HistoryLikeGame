# Culture Timeline Game Assets

Place event illustration images for the culture timeline game in this directory.

## Naming Convention

```
TL9_{event-slug}.jpg
```

The prefix `TL9` stands for Timeline Period 9 (Culture).

### Required Files

These slugs correspond to the events defined in `data/timeline/period-9-kultura.js`:

| File | Event |
|------|-------|
| `TL9_trypillia-ceramics.jpg` | Трипільська кераміка |
| `TL9_sophia-mosaics.jpg` | Мозаїки Софії Київської |
| `TL9_ostroh-bible.jpg` | Острозька Біблія |
| `TL9_cossack-baroque.jpg` | Козацьке бароко |
| `TL9_kotlyarevsky-eneida.jpg` | "Енеїда" Котляревського |
| `TL9_shevchenko-kobzar.jpg` | "Кобзар" Шевченка |
| `TL9_lysenko.jpg` | Микола Лисенко |
| `TL9_franko.jpg` | Іван Франко |
| `TL9_dovzhenko.jpg` | Олександр Довженко |
| `TL9_sixtiers.jpg` | Шістдесятники |
| `TL9_chornobyl-lit.jpg` | Чорнобильська тема в літературі |
| `TL9_modern-renaissance.jpg` | Сучасне відродження |

## Recommended Sizes

- Timeline thumbnails: 300×200 px (landscape)
- Used as small cards in the timeline drag-and-drop interface
- JPEG at 80% quality is sufficient
