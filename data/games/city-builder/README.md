# City Builder Game Assets

Place historical city building images in this directory.

## Required Files

The game engine expects these exact filenames:

| File | Description |
|------|-------------|
| `RATH_1.jpg` | Ратуша (Town Hall) |
| `TSER_1.jpg` | Церква / Собор (Church / Cathedral) |
| `SHKO_1.jpg` | Школа / Академія (School / Academy) |
| `FORT_1.jpg` | Фортеця / Замок (Fortress / Castle) |

## Optional Additional Files

Use `{CODE}_{n}.jpg` pattern for variants (n = 2, 3, ...):

```
RATH_2.jpg    — Alternative town hall style
TSER_2.jpg    — Baroque church variant
MARK_1.jpg    — Market / торговий майдан
MOST_1.jpg    — Bridge / міст
PORT_1.jpg    — City gate / міська брама
```

## Recommended Sizes

- All building images: 256×256 px or 512×512 px (square)
- Transparent PNG preferred for overlay on map backgrounds
- Use consistent art style (flat illustration or pixel art)
