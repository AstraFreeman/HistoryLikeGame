# Archaeology Game Assets

Place excavation site and artifact images in this directory.

## Naming Convention

```
{CULTURE}_{type}_{n}.{ext}
```

### Culture Codes

| Code | Culture |
|------|---------|
| TRPL | Трипільська культура |
| SKIF | Скіфська культура |
| GREK | Грецькі колонії |
| KAMN | Кам'яний вік |

### Type Values

- `site` — excavation site background
- `artifact` — individual artifact image
- `layer` — soil layer / cross-section image

### Examples

```
TRPL_artifact_1.jpg   — Трипільська кераміка
SKIF_artifact_1.jpg   — Скіфська пектораль
GREK_site_1.jpg       — Ольвія, розкопки
KAMN_layer_1.jpg      — Стратиграфічний розріз
```

## Recommended Sizes

- Site backgrounds: 800×600 px minimum
- Artifacts: 200×200 px (square, transparent PNG preferred)
- Layer images: 400×600 px
