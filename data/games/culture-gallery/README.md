# Culture Gallery Game Assets

Place cultural artwork and artifact images for the gallery game in this directory.

## Naming Convention

```
GAL_{era}_{n}.jpg
```

### Era Codes

| Code | Era | Period |
|------|-----|--------|
| TRPL | Трипільська доба | 5500–2750 до н.е. |
| KYIV | Київська Русь | IX–XIII ст. |
| BARO | Козацьке бароко | XVII–XVIII ст. |
| ROMA | Романтизм / XIX ст. | XIX ст. |
| MDRN | Сучасне мистецтво | XX–XXI ст. |

### Examples

```
GAL_TRPL_1.jpg    — Трипільська кераміка з розписом
GAL_TRPL_2.jpg    — Трипільська статуетка
GAL_KYIV_1.jpg    — Мозаїка Софії Київської
GAL_KYIV_2.jpg    — Ювелірні прикраси Русі
GAL_BARO_1.jpg    — Іконостас козацького бароко
GAL_ROMA_1.jpg    — Портрет роботи Шевченка
GAL_MDRN_1.jpg    — Сучасна українська живопис
```

## Recommended Sizes

- Gallery images: 400×300 px minimum (landscape orientation preferred)
- High quality JPEG at 85% compression
- Include diverse media: painting, sculpture, architecture, applied arts
